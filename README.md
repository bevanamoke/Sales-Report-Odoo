# Sales Store Expense Report

A comprehensive Odoo 18 module designed for detailed reporting of store expenses and sales, categorized by expense types. This module provides businesses with insightful analytics to track profitability and operational costs effectively.

## Features

- **Expense Tracking**: Categorize and monitor store expenses with ease.
- **Sales Integration**: Align sales data with corresponding expense categories for better financial clarity.
- **Dynamic Reporting**:
    - **PDF Reports**: Professional, print-ready PDF summaries.
    - **Excel Exports**: Detailed data sheets for further analysis.
- **Advanced UI Components**:
    - Custom Matrix Widgets for complex data visualization.
    - Interactive Product Category widgets.
- **Seamless Integration**: Works out-of-the-box with Odoo's `sale` and `account` modules.

## Installation

1. Copy the `sales_store_expense_report` folder to your Odoo custom addons directory.
2. Update the App List in your Odoo instance.
3. Install the "Sales Store Expense Report" module.

## Configuration

Ensure you have the necessary permissions to access sales and accounting reporting features.
